# 逆熵终端

## 起因

这个项目纯粹是基于我个人的需求和喜好开始的。

那时候每天累得要死，作息一塌糊涂。想着要是能有一个像玩游戏上瘾一样的东西来帮我养成习惯就好了。不知道为什么想到了日记——虽然我从来没这个习惯。然后觉得光写日记好无聊，很难坚持，要是能有互动就好了。跟喜欢的角色互动应该会好一点吧。

然后就开始了。

虽然最后作息也没救回来。

## 声明

本项目大量使用 AI 编程工具辅助开发，属于 AI 垃圾。仓库仅用于存档。

用过的主要有：

- **Google AI Studio**（早期）
- **OpenAI Codex CLI**
- **Claude Code**（接 DeepSeek V4 Pro）

从 2025 年 1 月 10 日发出第一条日志测试起，断断续续改到现在，2026 年才上传 GitHub。2026 年 2 月 22 日搞了个"主脑"的设定，但目前还没完善。

## 已有功能

- **赛博朋克风终端界面** — 游戏化 UI，SAN 值 / 能量 / 金币数值系统
- **AI 聊天** — 支持 Gemini / OpenAI / DeepSeek / 自定义 API，可切换不同人设
- **流式输出** — SSE 流式聊天，支持思考链（DeepSeek V4 reasoning）
- **毒舌通知** — Android 原生后台保活，定时推送 AI 生成的提醒（喝水/睡眠/运动/进食）
- **触觉反馈** — 根据 AI 消息中的情绪标记触发手机振动（[HAPTIC:emotion]）
- **生理日志** — 饮水/进食/睡眠/运动/起床记录，AI 自动分析
- **经期追踪** — 月经周期日历，流量/疼痛/情绪记录
- **专注模式** — 番茄钟 + 失败嘲讽
- **抽奖系统** — 金币消耗，随机道具
- **记忆库** — 全局记忆存档 + 手动总结
- **数据备份/恢复** — JSON 导出导入
- **Android 原生桥接** — 后台服务、通知、振动、流式 SSE 均由 Java 层实现
- **多连接配置** — 可分别为聊天和通知配置不同的 AI 提供商，支持自定义代理地址

## 还没做 / 做了一半

- 主脑（Overseer）设定仅有壳，逻辑没完善
- 记忆系统缺少自动压缩、启发式搜索、用户画像定期更新
- 人设切换时记忆归属逻辑没理清
- 聊天消息长按操作菜单（多选删除/复制/编辑/重roll）
- 思维链不是流式输出
- AI 幻觉问题
- 工具调用太慢，需要省 token
- 页面排版需要重新设计
- 情感 Live2D
- 长对话/短对话模式切换
- AI 意愿算法
- 除睡眠外其他提醒未记录到通知收件箱

## 环境配置

在项目根目录创建 `.env` 文件：

```env
API_KEY=你的API_Key
```

或者 `GEMINI_API_KEY=你的Key`（vite 构建时会自动读取）。

## 本地运行

```bash
npm install
npm run dev
```

## Android 编译

用 Android Studio 打开 `android/` 目录，sync 后直接 Build。注意 `capacitor.config.ts` 里的 `hostname` 是瞎写的，不影响本地打包。
